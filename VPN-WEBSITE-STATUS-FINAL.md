## 🏃 Quick Start (Choose Your Path)

### Path 1: Run Website Only

```bash
npm install
npm start
# Open: http://localhost:8080 ✅
```

### Path 2: Run Website + VPN

```bash
./generate_wg_config.sh          # Generate WG keys
./generate_dilithium_keys.sh     # Generate PQC keys
npm start                         # Start server
sudo wg-quick up ./wg0.conf      # Start VPN
# Access: http://localhost:8080 + VPN on 51820 ✅
```

### Path 3: Run in Docker

```bash
docker build -t rmada .
docker run -p 8080:8080 -p 51820:51820/udp rmada
# Access: http://localhost:8080 ✅
```

### Path 4: Run with Earthly

```bash
earthly +docker-build
earthly +export-tarball
# Get portable image ✅
```

## 🎓 How It Works (TL;DR)

```
1. WEBSITE (you open in browser)
   └─ Shows 6 real-time charts of device data
   └─ Connects via WebSocket for live updates
   └─ Secured with JWT authentication

2. VPN (devices connect to)
   └─ WireGuard protocol on port 51820
   └─ Devices sign with Dilithium (post-quantum safe)
   └─ Server verifies and assigns IP 10.0.0.x
   └─ All traffic encrypted end-to-end

3. DATABASE (persists everything)
   └─ SQLite stores users, devices, telemetry, sessions
   └─ Survives server restarts
   └─ Auto-backups every hour

4. SECURITY
   └─ Passwords hashed with bcryptjs
   └─ JWT tokens expire after 1 hour
   └─ Dilithium post-quantum verification
   └─ OpenSSL HTTPS certificates
   └─ WireGuard IP isolation (10.0.0.0/24)

Result: Production-ready real-time monitoring system!
```
