📝 COMO TESTAR MANUALMENTE

1. TESTE NEGATIVO (Deve falhar)
   $ curl http://localhost:8080/api/devices
   Resultado: 403 - {"error":"authentication required"}

2. REGISTRAR PROPRIETÁRIO
   $ curl -X POST http://localhost:8080/api/register-owner \
     -d '{"username":"seu_user","password":"Senha@123","ownerCode":"..."}'
   Resultado: {"token":"eyJ...","role":"owner"}

3. USAR TOKEN
   $ TOKEN="eyJ..."
   $ curl -H "Authorization: Bearer $TOKEN" \
     http://localhost:8080/api/devices
   Resultado: [{"id":"D1",...}]

4. DEFESA CIVIL NÃO PODE ENVIAR
   $ curl -X POST -H "Authorization: Bearer $DEFENSE_TOKEN" \
     http://localhost:8080/api/telemetry \
     -d '{"deviceId":"D1","value":42}'
   Resultado: 403 - {"error":"only owners can submit telemetry"}
